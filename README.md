# PulmoRehab Starter Kit

An open-source starter toolkit for pulmonary rehabilitation education, tracking, and simple clinical workflow support.

## Overview
PulmoRehab Starter Kit is a lightweight open-source project designed to support pulmonary rehabilitation education, patient tracking, and simple rehabilitation-oriented clinical workflows.

This project aims to make practical rehabilitation resources more accessible to students, clinicians, researchers, and developers who want to build useful tools for pulmonary care and perioperative recovery.

## Why this project exists
Pulmonary rehabilitation and perioperative recovery often rely on fragmented educational materials, inconsistent tracking methods, and limited lightweight digital tools. This repository is created to provide openly accessible, reusable resources that can gradually grow into a more practical toolkit.

## Current scope
This repository currently focuses on:
- pulmonary rehabilitation tracking templates
- patient education materials
- simple data organization scripts
- starter documentation for future development
